# Parallax Shop (Next.js + Supabase)

## ENV (Vercel / локально)
- NEXT_PUBLIC_SUPABASE_URL
- NEXT_PUBLIC_SUPABASE_ANON_KEY
- NEXT_PUBLIC_SITE_URL (например https://parallax-shop.vercel.app) — для sitemap

## База
Выполни `supabase.sql` в Supabase -> SQL Editor.

## Запуск
```bash
npm i
npm run dev
```
